# My-Tennis-Club how to run
Step 1 : Download django using pip in cmd. 
Step 2 : Download files. 
Step 3 : Activate django Virtual environment. as myworld\Scripts\activate.bat
Step 4 : Make migrations of files. as py manage.py migrate
Step 5 : Run program as py manage.py runserver. 
Step 6 : Copy the url and paste on browser. 
Step 7 : To create super user as py manage.py createsuperuser. 
Step 8 : To get access to database type '\admin' on url.
